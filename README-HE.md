# מעגל גברים

דף נחיתה למעגל גברים -- מסע ללב הגבריות. עיצוב מודרני בעברית עם אנימציות חלקות ותצוגה מותאמת לכל מכשיר.

## תכונות

- דף נחיתה מעוצב בעברית מלאה
- אנימציות גלילה חלקות עם Framer Motion
- עיצוב מודרני ונקי עם Tailwind CSS
- אנליטיקס של Vercel
- עיצוב מותאם מובייל

## טכנולוגיות

| טכנולוגיה | שימוש |
|-----------|-------|
| [Next.js 16](https://nextjs.org/) | פריימוורק React |
| [React 19](https://react.dev/) | ספריית UI |
| [TypeScript](https://www.typescriptlang.org/) | בטיחות טיפוסים |
| [Tailwind CSS 4](https://tailwindcss.com/) | עיצוב |
| [Framer Motion](https://www.framer.com/motion/) | אנימציות |
| [Vercel Analytics](https://vercel.com/analytics) | אנליטיקס |

## התקנה

```bash
git clone https://github.com/eladjak/mens-circle.git
cd mens-circle
bun install
bun dev
```

פתחו [http://localhost:3000](http://localhost:3000) בדפדפן.
