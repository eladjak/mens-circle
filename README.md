# Men's Circle

A modern landing page for a men's personal growth circle (Ma'agal Gvarim), featuring an engaging Hebrew RTL design with smooth animations and responsive layout.

## Features

- Responsive Hebrew RTL landing page
- Smooth scroll animations with Framer Motion
- Modern, clean UI with Tailwind CSS
- Vercel Analytics integration
- Mobile-first design

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| [Next.js 16](https://nextjs.org/) | React framework |
| [React 19](https://react.dev/) | UI library |
| [TypeScript](https://www.typescriptlang.org/) | Type safety |
| [Tailwind CSS 4](https://tailwindcss.com/) | Utility-first styling |
| [Framer Motion](https://www.framer.com/motion/) | Animations |
| [Vercel Analytics](https://vercel.com/analytics) | Visitor analytics |

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (recommended) or Node.js 20+

### Installation

```bash
git clone https://github.com/eladjak/mens-circle.git
cd mens-circle

bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
bun run build
bun start
```
